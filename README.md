## 🎯 **Project Overview**

This project simulates **5 days in the life of an L2 SOC Analyst**. I built a complete security monitoring lab using **Wazuh**, created **5 custom detection rules**, and investigated each alert using professional L2 methodology.

### **The Scenario**
> As an L2 SOC Analyst, I receive daily alerts from my Wazuh SIEM. Each day, I investigate a different type of threat, following the "L2 Five" framework to understand the full attack chain, extract IOCs, and provide actionable recommendations.

### **What Makes This Project Different**
- ✅ **Real-world investigation methodology** - Not just "alert fired"
- ✅ **Full attack chain analysis** - Initial access to exfiltration
- ✅ **Actionable recommendations** - GPOs, logging, hardening
- ✅ **Professional reporting** - Executive to technical levels
- ✅ **Telegram integration** - Real-time SOC alerting
